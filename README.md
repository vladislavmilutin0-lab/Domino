# 🎲 Domino Fall — Unity Project
 
Минималистичный симулятор цепной реакции падения доминошек, реализованный на Unity. В проекте **нет клавиш, кнопок или UI**: симуляция физики запускается автоматически сразу после нажатия ▶ **Play** в редакторе. Проект подходит как база для тестов PhysX, демонстрации работы с коллайдерами или учебного примера.
 
## ✨ Возможности
- Автоматический запуск симуляции при старте сцены
- Реалистичная физика падения, столкновений и затухания
- Настройка параметров через Prefabs и `Physics Material`
- Готов к использованию без дополнительных скриптов управления
 
## 🛠 Требования
- **Unity:** 2022.3 LTS или новее
- **C#:** 10+
- **ОС:** Windows / macOS / Linux
- **Минимальные требования:** GPU с поддержкой OpenGL 4.5 / DirectX 11, 4 GB RAM
 
## 🚀 Как запустить
1. Клонируйте репозиторий: `git clone <url>`
2. Откройте папку проекта через Unity Hub
3. В редакторе перейдите в `Assets/Scenes` и откройте сцену
4. Нажмите ▶ **Play** — доминошки начнут падать автоматически
 
> ⚠️ В проекте **не привязаны клавиши или элементы управления**. Симуляция работает только в режиме Play редактора или в собранном билде (сцена стартует сразу при запуске приложения).
 
 
## 💡 Советы по стабильной физике
Если доминошки «зависают», падают рывками или не запускают цепную реакцию:
- Уменьшите `Fixed Timestep` (`Edit → Project Settings → Time`) до `0.01`–`0.016`
- В `Project Settings → Physics` увеличьте `Solver Iterations` до `10–15`
- Проверьте `Physics Material`: `Dynamic Friction ≈ 0.3`, `Bounciness ≈ 0.05–0.1`
- Убедитесь, что у всех объектов включён `Use Gravity` и масса `Mass` находится в диапазоне `0.5–1`
 
## 📦 Сборка (Build)
1. `File → Build Settings`
2. Перетащите текущую сцену в список
3. Выберите платформу (рекомендуется **PC, Mac & Linux Standalone**)
4. Нажмите `Build` → запустите полученный `.exe` / приложение
